# BizCrush Desktop

BizCrush desktop application distribution repository.

## Download

Download the latest version:
- **macOS**: [Download](https://github.com/BizCrush/BizCrush/releases/latest)
- **Windows**: [Download](https://github.com/BizCrush/BizCrush/releases/latest)

## Auto-Updates

The app uses Sparkle (macOS) and WinSparkle (Windows) for automatic updates.

Update feeds:
- macOS: https://bizcrush.github.io/BizCrush/downloads/macos/appcast.xml
- Windows: https://bizcrush.github.io/BizCrush/downloads/windows/appcast.xml

## Installation

### macOS
1. Download the ZIP file
2. Extract and move BizCrush.app to Applications folder
3. On first launch, right-click and select "Open" to bypass Gatekeeper

### Windows
1. Download the ZIP file
2. Extract to a location of your choice
3. Run BizCrush.exe
4. Windows may show SmartScreen warning - click "More info" and "Run anyway"
