Installation

  1. Download the installer (BizCrush-Setup-0.22.5.exe)
  2. Run the installer and follow the setup wizard
  3. Windows may show a SmartScreen warning - click "More info" and "Run anyway"
  4. The app will be installed to C:\Program Files\Biz-Crush\
  5. Desktop and Start Menu shortcuts will be created automatically

System Requirements

  • Windows 10 version 1809 or later
  • Windows 11 (recommended)
  • 64-bit processor
